第四章（这章是编程的重点）

条件转移指令

助记符 标志位 中文说明
JZ/JE ZF = 1 等于零/想等
JNZ/JNE ZF = 0 不等于零/不想等
JS SF = 1 符号为负
JNS SF = 0 符号为正
JC/JB/JNAE CF = 1 进位/低于/不高于等于
JNC/JNB/JAE CF = 0 无进位/不低于/高于等于
JBE/JNA CF = 1 或 ZF = 1 低于等于/不高于
JNBE/JA CF = 0 且 ZF = 0 不低于等于/高于
JL/JNGE SF ≠ OF 小于/不大于等于
JNL/JGE SF = OF 不小于/大于等于
JLE/JNG SF ≠ OF 或 ZF = 1 小于等于/不大于
JNLE/ JG SF = OF 且 ZF = 0 不小于等于/大于

TEST
